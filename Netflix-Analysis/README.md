# 🎬 Netflix Data Analysis using SQL

## 📌 Objective
The goal of this project is to analyze Netflix dataset and understand content distribution, trends, and popular categories.

## 🛠 Tools Used
- SQL

## 📂 Dataset
- Netflix dataset (netflix.csv)

## 📊 Analysis Performed
- Total content count
- Movies vs TV Shows distribution
- Top countries producing content
- Content trend over years
- Most popular genres

## 💡 Key Insights
- Movies dominate Netflix content compared to TV shows
- Few countries contribute majority of content
- Content production increased over the years
- Certain genres are more popular

## 📁 Files Included
- netflix_analysis.sql
- netflix.csv

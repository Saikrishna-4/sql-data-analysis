# 📊 Sales Data Analysis using SQL

## 📌 Objective
Analyze retail sales data to identify customer behavior, product performance, and regional trends.

## 🛠 Tools Used
- SQL

## 📂 Dataset
- Superstore dataset (superstore.csv)

## 📊 Analysis Performed
- Total orders and sales
- Unique customers
- Top regions by sales
- Highest selling products
- Monthly sales trends

## 💡 Key Insights
- Few customers contribute major revenue
- Certain regions dominate sales
- Some products perform significantly better
- Sales vary across months

## 📁 Files Included
- sales_analysis.sql
- superstore.csv

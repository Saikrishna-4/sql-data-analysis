# 📊 Sales Data Analysis using SQL

## 📌 Objective
The goal of this project is to analyze a retail sales dataset and extract meaningful insights using SQL.

## 🛠 Tools Used
- SQL

## 📂 Dataset
- Superstore sales dataset (superstore.csv)

## 📊 Analysis Performed
- Total Sales Calculation
- Top 5 Customers by Revenue
- Region-wise Sales Analysis
- Top 10 Products by Sales
- Monthly Sales Trends

## 💡 Key Insights
- A small number of customers contribute to a large portion of revenue
- Certain regions generate higher sales compared to others
- Some products dominate overall sales performance
- Sales vary across different months

## 📁 Files Included
- sales_analysis.sql
- superstore.csv
